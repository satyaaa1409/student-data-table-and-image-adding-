#  Student Data Table Project

This is a simple HTML & CSS project that displays student data in a structured table format.
Adding image using html.
It is created as a beginner project to practice basic web development concepts.

## Technologies Used
* HTML
* CSS
  
Displays:

* Student Name, Age, and City in a table
* A styled heading
* An image section below the table
* 
##  What I Learned

* Creating tables using `<table>`, `<tr>`, `<th>`, `<td>`
* Styling with CSS (colors, spacing, hover effects)
* Linking external CSS file
* Basic HTML page structure
